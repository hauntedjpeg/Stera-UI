## Installation

```bash
pnpm dlx stera-ui add pagination
```

## Usage

```tsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationPrevious,
  PaginationNext,
  PaginationEllipsis,
} from "@/components/ui/pagination"
```

```tsx
<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="#" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#" isActive>1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="#" />
    </PaginationItem>
  </PaginationContent>
</Pagination>
```

## API Reference

### PaginationLink

| Prop | Type | Default |
| --- | --- | --- |
| `href` | `string` | — |
| `isActive` | `boolean` | `false` |

### PaginationPrevious / PaginationNext

| Prop | Type | Default |
| --- | --- | --- |
| `href` | `string` | — |
| `text` | `string` | `"Previous"` / `"Next"` |
