## Composition

Use the following composition to build a `Pagination`:

```text
Pagination
└── PaginationContent
    ├── PaginationItem
    │   └── PaginationPrevious
    ├── PaginationItem
    │   └── PaginationLink
    ├── PaginationItem
    │   └── PaginationEllipsis
    └── PaginationItem
        └── PaginationNext
```

## API Reference

### PaginationLink

| Prop | Type | Default |
| --- | --- | --- |
| `href` | `string` | — |
| `isActive` | `boolean` | `false` |

### PaginationPrevious / PaginationNext

| Prop | Type | Default |
| --- | --- | --- |
| `href` | `string` | — |
| `text` | `string` | `"Previous"` / `"Next"` |
