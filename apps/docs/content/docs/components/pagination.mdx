### PaginationLink

| Prop | Type | Default |
| --- | --- | --- |

### PaginationPrevious / PaginationNext

| Prop | Type | Default |
| --- | --- | --- |
| `href` | `string` | — |
| `text` | `string` | `"Previous"` / `"Next"` |
