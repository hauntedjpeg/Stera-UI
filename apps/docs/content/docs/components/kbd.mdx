## API Reference

### Kbd

| Prop | Type | Default |
| --- | --- | --- |
| `size` | `"sm" \| "md"` | `"md"` |

Renders a `<kbd>` element. Accepts all standard `<kbd>` props.

### KbdGroup

Renders a `<kbd>` wrapper for grouping multiple `Kbd` components. Accepts all standard `<div>` props.
