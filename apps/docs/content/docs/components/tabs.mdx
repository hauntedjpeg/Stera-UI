## API Reference

### Tabs

| Prop | Type | Default |
| --- | --- | --- |
| `defaultValue` | `string` | — |
| `value` | `string` | — |
| `onValueChange` | `(value: string) => void` | — |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"default" \| "line"` | `"default"` |
