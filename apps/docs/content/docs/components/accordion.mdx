## Installation

```bash
pnpm dlx stera-ui add accordion
```

## Multiple Open

<ComponentPreview name="accordion-multiple" previewClassName="*:data-[slot=accordion]:max-w-sm" />

Pass `multiple` to allow more than one item to be expanded at a time.

## Default Open

<ComponentPreview name="accordion-default-open" previewClassName="*:data-[slot=accordion]:max-w-sm" />

Use `defaultValue` with an array of item values to pre-expand items on first render.

## API Reference

### Accordion

| Prop | Type | Default |
| --- | --- | --- |
| `multiple` | `boolean` | `false` |
| `defaultValue` | `string[]` | `[]` |
| `value` | `string[]` | — |
| `onValueChange` | `(value: string[]) => void` | — |

### AccordionItem

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | — |
| `disabled` | `boolean` | `false` |
