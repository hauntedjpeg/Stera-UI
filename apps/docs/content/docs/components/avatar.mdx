<AvatarDefault />

## Installation

```bash
pnpm dlx stera-ui add avatar
```

## Usage

```tsx
import {
  Avatar,
  AvatarFallback,
  AvatarImage,
} from "@/components/ui/avatar"
```

```tsx
<Avatar>
  <AvatarImage src="/avatar.jpg" alt="Jane Appleseed" />
  <AvatarFallback>JA</AvatarFallback>
</Avatar>
```
