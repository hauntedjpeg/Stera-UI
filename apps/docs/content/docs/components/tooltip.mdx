## Installation

```bash
pnpm dlx stera-ui add tooltip
```

## Sides

<ComponentPreview name="tooltip-sides" />

| Side | Description |
| --- | --- |
| `top` | Above the trigger (default) |
| `right` | Right of the trigger |
| `bottom` | Below the trigger |
| `left` | Left of the trigger |

<ComponentPreview name="tooltip-with-kbd" />

## API Reference

### TooltipProvider

| Prop | Type | Default |
| --- | --- | --- |
| `delayDuration` | `number` | `700` |

### TooltipContent

| Prop | Type | Default |
| --- | --- | --- |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"top"` |
| `align` | `"start" \| "center" \| "end"` | `"center"` |
| `sideOffset` | `number` | `4` |
| `alignOffset` | `number` | `0` |

### TooltipTrigger

| Prop | Type | Default |
| --- | --- | --- |
| `asChild` | `boolean` | `false` |
