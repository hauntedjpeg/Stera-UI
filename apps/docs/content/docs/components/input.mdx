## API Reference

Built on Base UI's [Input](https://base-ui.com/react/components/input#api-reference) primitive — see the full API reference there. Accepts all standard `<input>` props except `size`.

| Prop | Type | Default |
| --- | --- | --- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` |
| `type` | `string` | `"text"` |
