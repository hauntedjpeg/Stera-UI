title: Toggle Group
description: A set of two-state buttons that can be toggled on or off individually or exclusively.
---

<ToggleGroupDefault />

## Installation

```bash
pnpm dlx stera-ui add toggle-group
```

## Usage

```tsx
import { ToggleGroup, ToggleGroupItem } from "@/components/ui/toggle-group"
```

```tsx
<ToggleGroup multiple>
  <ToggleGroupItem value="bold" aria-label="Toggle bold">
    <SiTextBold />
  </ToggleGroupItem>
  <ToggleGroupItem value="italic" aria-label="Toggle italic">
    <SiTextItalic />
  </ToggleGroupItem>
</ToggleGroup>
```

## API Reference

### ToggleGroup

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"default" \| "outline"` | `"default"` |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "icon" \| "icon-xs" \| "icon-sm" \| "icon-lg"` | `"icon"` |
| `multiple` | `boolean` | `false` |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |
| `spacing` | `number` | `0` |
| `disabled` | `boolean` | `false` |

### ToggleGroupItem

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | — |
| `disabled` | `boolean` | `false` |
