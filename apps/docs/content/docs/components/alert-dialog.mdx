## API Reference

### AlertDialogContent

| Prop | Type | Default |
| --- | --- | --- |
| `size` | `"default" \| "sm"` | `"default"` |

### AlertDialogAction

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"default" \| "danger"` | `"default"` |

### AlertDialogTrigger

| Prop | Type | Description |
| --- | --- | --- |
| `render` | `ReactElement` | Render prop for the trigger element |
