## API Reference

Built on Base UI's [Switch](https://base-ui.com/react/components/switch#api-reference) primitive — see the full API reference there.

| Prop | Type | Default |
| --- | --- | --- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` |
| `checked` | `boolean` | — |
| `defaultChecked` | `boolean` | — |
| `onCheckedChange` | `(checked: boolean) => void` | — |
| `disabled` | `boolean` | `false` |
| `name` | `string` | — |
| `value` | `string` | — |
