title: Switch
description: A toggle control for switching between two states, like on and off.
---

<SwitchDefault />

## Installation

```bash
pnpm dlx stera-ui add switch
```

## Usage

```tsx
import { Switch } from "@/components/ui/switch"
```

```tsx
<Switch />
```

## With Label

Pair with `Field` and `FieldLabel` for an accessible labeled switch.

<SwitchDefault />

```tsx
<Field orientation="horizontal">
  <Switch id="airplane-mode" />
  <FieldLabel htmlFor="airplane-mode">Airplane Mode</FieldLabel>
</Field>
```

## With Description

Place `FieldContent` between the label and switch to add a description.

<SwitchWithDescription />

```tsx
<Field orientation="horizontal">
  <FieldContent>
    <FieldLabel htmlFor="marketing">Marketing emails</FieldLabel>
    <FieldDescription>
      Receive emails about new products, features, and more.
    </FieldDescription>
  </FieldContent>
  <Switch id="marketing" />
</Field>
```

| Prop | Type | Default |

| `size` | `"default" \| "sm"` | `"default"` |

| `defaultChecked` | `boolean` | — |
| `onCheckedChange` | `(checked: boolean) => void` | — |
| `disabled` | `boolean` | `false` |
| `name` | `string` | — |
| `value` | `string` | — |
