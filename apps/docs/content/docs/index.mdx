title: Introduction
description: A component registry for React, built on Base UI and Tailwind CSS.
---

Stera UI is a component registry. The CLI copies component source into your project — there is no runtime package to install or version to track.

Components are built on [Base UI](https://base-ui.com) primitives, styled with [Tailwind CSS v4](https://tailwindcss.com), and use [stera-icons](https://www.npmjs.com/package/stera-icons) for iconography. Theming is handled through CSS custom properties; dark mode is included.

## Quick start

<Steps>

### Initialize

```bash
pnpm dlx stera-ui init
```

### Add components

```bash
pnpm dlx stera-ui add button dialog
```

### Use them

```tsx
import { Button } from "@/components/ui/button"

export function MyPage() {
  return <Button variant="brand">Click me</Button>
}
```

</Steps>
