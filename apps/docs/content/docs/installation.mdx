title: Installation

---

## Requirements

- React project (Next.js or Vite)
- Tailwind CSS v4
- TypeScript (recommended)

## Initialize

```bash
pnpm dlx stera-ui init
```

`init` detects your project structure, writes `components.json`, creates `stera-ui.css` with the design tokens, and installs `tw-animate-css`.

## Add components

```bash
pnpm dlx stera-ui add button
```

Multiple at once:

```bash
pnpm dlx stera-ui add button dialog dropdown-menu
```

Registry dependencies are resolved automatically — adding `dialog` also pulls in `button` and `utils` if missing.

### Flags

- `--yes` — skip prompts and accept defaults
- `--overwrite` — replace existing files without asking

## Refresh design tokens

```bash
pnpm dlx stera-ui add globals
```

Re-pulls the latest `stera-ui.css` from the registry. Use this to pick up token updates without re-adding components.

## List components

```bash
pnpm dlx stera-ui list
```

Groups components by type: `ui`, `component`, `font`, `hook`, `lib`, `style`.

## Project structure

After `init`:

```text noLineNumbers
your-project/
├── components.json          # Stera UI configuration
└── styles/
    ├── globals.css          # Imports stera-ui.css + peer styles
    └── stera-ui.css         # Design tokens + utilities
```

After your first `stera-ui add`:

```text noLineNumbers
your-project/
├── components.json
├── components/
│   └── ui/                  # Installed components
│       └── button.tsx
├── lib/
│   └── utils.ts             # cn() utility
└── styles/
    ├── globals.css
    └── stera-ui.css
```
